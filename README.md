# Version Everything: From Chaos to Order in Reproducible Python Projects

**EuroSciPy 2026 — Workshop**

[EuroSciPy Schedule](https://euroscipy.org/schedule/)

## About

This hands-on workshop tackles the "works on my machine" problem by teaching reproducible workflow practices. Participants learn to lock dependencies, version control code and data, externalize parameters, containerize applications, and implement collaboration practices.

**Track:** Computational Tools and Scientific Python Infrastructure

## Agenda

- Modern Dependency Management
- Code & Configuration Versioning
- Data Pipeline Versioning
- Hidden Reproducibility Challenges
- Production Deployment

## Prerequisites

- Admin access on your laptop for tool installation
- Basic Python and command-line knowledge

## About the Speaker

Aris Nivorlis is a researcher geophysicist and data steward at Deltares, focused on promoting good practices in data management and scientific coding.
